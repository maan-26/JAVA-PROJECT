# Attendance-Tracker-
Attendance Strategist is a comprehensive offline application for tracking and projecting academic attendance. It integrates customizable weekly timetables, dynamic calendar exceptions for holidays and variable weekend schedules, and a predictive calculator to precisely determine the future classes required to meet a target percentage.


### 1. SYSTEM REQUIREMENTS
It is required that you have the Java Development Kit (JDK) installed in your system.
To verify the presence of the JDK in your system, you can use the following command in your command prompt:
java -version

### 2. COMPILATION
To compile the source code, you can use the following command in your command prompt:
javac AttendanceApp.java

### 3. EXECUTION
To run the program, you can use the following command in your command prompt:
java AttendanceApp

### 4. DATA PERSISTENCE
Once the program is executed, it will automatically generate a file named 'attendance_data.ser' in the root directory of the project.
It is recommended that you maintain this file to preserve your data.
If you delete the file, it will be treated as a complete factory reset of the application.
